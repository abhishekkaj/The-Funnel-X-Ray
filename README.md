# The Funnel X-Ray

## Overview
The Funnel X-Ray is a modern Manifest V3 Chrome Extension designed as a one-click diagnostic tool to dissect the digital marketing strategy of any active webpage.

## Features
- **Architecture (WP) Analysis:** Detects WordPress installations and extracts active themes and plugins.
- **SEO Diagnostic Engine:** Analyzes Title, Meta Description, Canonical URL, Heading tags (H1/H2), and calculates keyword density of the top 5 organic terms using advanced stop-word filtering.
- **Facebook Ads & Pixel Inspector:** Spots active Facebook Pixels on the page and provides a one-click link to the Facebook Ads Library, pre-filtered for the domain.

## Manual Installation Instructions
1. Download or locate this project folder (`The Funnel X-Ray`) on your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** by toggling the switch in the top right corner.
4. Click the **"Load unpacked"** button that appears in the top left.
5. Select the `The Funnel X-Ray` folder containing the `manifest.json` file.
6. The extension is now installed! Pin it to your browser toolbar for easy access.

## Usage
Navigate to any webpage and click the extension icon to instantly generate a digital marketing diagnostic report. The extension uses a premium dark mode UI and supports dynamic content parsing.
